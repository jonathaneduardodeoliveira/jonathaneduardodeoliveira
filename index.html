Jogo da Velha em Java

Este é um projeto de um Jogo da Velha desenvolvido em Java. O jogo foi criado com funcionalidades que permitem que dois jogadores se enfrentem em um tabuleiro clássico de 3x3. O jogo verifica automaticamente as jogadas e determina o vencedor ou empate. O código foi escrito utilizando boas práticas de programação, e o projeto foi organizado para ser facilmente compreensível e extensível.
